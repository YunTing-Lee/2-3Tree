# 2-3Tree

## 開發平台
Windows 10

## 開發環境
Code::Block20.03

## input檔案格式
- 前3列為標題  
- 第4列開始每行依序為[學校代碼][學校名稱][科系代碼][科系名稱][日間∕進修別][等級別][學生數][教師數][上學年度畢業生數][縣市名稱][體系別][，不同欄位之間以"tab"隔開  


## 說明 : 
將input檔案中的學校資訊以**學校名稱**為鍵值一筆一筆新增至23樹中，最後輸出樹高以及根根節點內所儲存的學校資訊
